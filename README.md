Three.js Basics
A simple React project demonstrating basic 3D rendering using Three.js with React Three Fiber.
It shows how to set up a 3D scene, add objects, lighting, and camera controls inside a React application.
________________________________________
Installation & Setup

Clone the repository
git clone
cd three.js-class-main

Install dependencies
npm install

Start the development server
npm run dev

Open in browser:
http://localhost:5173
