Three.js React Basics
A React-based project demonstrating basic 3D rendering using Three.js integrated with React through React Three Fiber. The project focuses on setting up a 3D scene, rendering objects, applying lighting, and enabling camera controls within a modern React application.
________________________________________
Project Overview
This project serves as an introductory example for using Three.js in React applications. It illustrates how 3D graphics can be rendered and managed declaratively using React components.
________________________________________
Tech Stack
•	React
•	Three.js
•	@react-three/fiber
•	@react-three/drei
•	Vite
________________________________________
Installation & Setup
Follow the steps below to run the project locally.
1. Clone the Repository
git clone <repository-url>
cd three.js-class-main
2. Install Dependencies
npm install
3. Start the Development Server
npm run dev
4. Open in Browser
http://localhost:5173
________________________________________
Usage
•	Launch the application in the browser
•	Interact with the 3D scene using mouse controls
•	Observe object rendering, lighting, and camera movement
